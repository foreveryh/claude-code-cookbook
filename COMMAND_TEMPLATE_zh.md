### 用法

```bash
/command-name [选项]
```

### 选项

- 无 : 默认行为
- `--option1` : 选项 1 的描述
- `--option2 <值>` : 带参数选项的描述
- `--flag` : 标志选项的描述

*注：如果命令没有选项，可以省略此章节。*

### 基本示例

```bash
# 基本用法
/command-name
"执行时的说明"

# 带选项的示例
/command-name --option1
"使用选项 1 时的说明"

# 带参数选项的示例
/command-name --option2 value
"使用带参数选项的示例"

# 组合多个选项
/command-name --option1 --flag
"组合多个选项的示例"
```
